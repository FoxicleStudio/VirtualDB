Virtual Database Implemented in C

Please use the VirtualDatabase2.3_Installer.exe to install the software.

The folder "db" must be along side of VirtualDB.exe at all times. If the folder is misplaced, the software will malfunction.

This software is released under GNU OpenSource license.

You can access the sourcecode and further updates at http://www.github.com/FoxicleStudio/VirtualDB/
